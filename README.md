## 👨‍💻 Self-hosting (For Devs)
```
# Install Git First (apt-instll git)
$ git clone https://github.com/szsupunma/sz-rose-bot
$ cd szrosebot
# Upgrade sources
$ bash deploy.sh
# Install All Requirements 
$ pip3 install -r requirements.txt
# Start Bot 
$ python3 -m DewmiBot
```
